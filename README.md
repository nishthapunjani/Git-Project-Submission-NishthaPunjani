# Git Project Submission: Python Calculator

A simple command-line calculator built to demonstrate **Git workflow**, **branching strategies**, and **conflict resolution** as part of the Final Project requirements.

## 🚀 Features
- **Basic Arithmetic:** Addition, Subtraction, Multiplication, and Division.
- **Advanced Math:** Power (exponentiation) functionality.
- **Error Handling:** Robust check for Division by Zero and invalid numerical inputs.
- **Interactive Menu:** Loop-based interface with a clean exit option.

## 🛠️ Git Workflow Demonstrated
This project follows a specific multi-developer branch simulation:
1.  **Main Branch:** The stable production code.
2.  **Developer1 Branch:** Initial feature updates and enhancements.
3.  **Developer2 Branch:** Parallel development used to induce and resolve a **merge conflict**.
